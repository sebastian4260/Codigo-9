Elementos de los formularios

Lista de Elementos

Nro Nombre del elemento Descripcion Ejemplo
1 input Es un campo de entrada que por medio de su atributo type se puede estipular que tipo de info. requerimos
2 textarea Es un campo de entrada que puede terner multilinea y se define sus dimensiones mediante las propiedades cols y rows
3 select
  • Es un elemento que contiene una lista de opciones a elegir
  • Cada opcion esta contenida dentro de otra etiqueta llamada option
4 button Elemento que representa a un boton, tiene las mismas caracteristicas que un input:button pero el button tiene etiqueta de apertura y cierre
5 label Es un elemento en el cual se recomienda poner el texto a mostrar para que la pagina sea mejor indexada y mediante su propiedad for al hacer click en el label se seleccionara su input
6 fieldset Es un elemento en el cual se agrupa otros elementos
Login
7 legend Representa al titulo de algunas etiquetas, por lo general va de la mano con el fieldset
